Política de privacidade

Última atualização: 15 de julho de 2026.

A Tábua de Marés API fornece dados de maré e recursos de integração. Coletamos apenas os dados necessários para operar a conta, autenticar o acesso, controlar o uso da API, processar assinaturas e atender solicitações de suporte. Quando você entra com o Google, recebemos os dados básicos autorizados por você, como nome, e-mail e imagem de perfil, para criar e manter sua conta.

Usamos esses dados para prestar o serviço, proteger a API, aplicar limites de uso, processar pagamentos e cumprir obrigações legais. Não vendemos dados pessoais. Compartilhamos dados somente com provedores necessários à operação, como Google para autenticação, Stripe para pagamentos e provedores de infraestrutura e banco de dados, sempre conforme a finalidade do serviço.

As credenciais de API e os dados de conta devem ser mantidos em segurança. Mantemos medidas técnicas e administrativas razoáveis, mas nenhum serviço conectado à Internet é completamente livre de riscos. Retemos os dados enquanto a conta estiver ativa ou pelo período necessário para cumprir obrigações legais, resolver disputas e prevenir abusos.

Você pode solicitar acesso, correção ou exclusão dos seus dados, respeitadas as obrigações legais e os registros necessários à segurança e faturamento. Para dúvidas ou solicitações, escreva para contato@devtu.qzz.io. Alterações relevantes nesta política serão publicadas nesta página.
